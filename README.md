# **Sakenny** – UI/UX Graduation Project

---

## Project Idea
**Sakenny** is a digital platform designed to simplify the housing search experience, especially for individuals living away from home.  
The platform provides suitable accommodation options based on users’ preferences, budget, and location — all through an intuitive and user-friendly interface.

## Work Plan

### 1. Research & Analysis
Conducting user research to identify challenges in the housing search process and understand market needs.

### 2. Audience Personas
Creating personas based on collected data to define user segments and their behaviors.

### 3. Visual Identity
Developing the brand’s visual direction including typography, layout style, and design tone.

### 4. Logo Design
Designing the project logo and defining the visual identity.

**Logo Drive Link:**  
(https://drive.google.com/file/d/1f5yQT90DomqrTBEq_2ane10o6WPnZXuJ/view?usp=sharing)

### 5. Main Designs
Designing the app and web interfaces using **Figma**, including:
- Screens  
- Design System  
- User Flow  
- Wireframes 

### 6. Complementary Products
Creating additional visual elements that support the brand identity.

### 7. Review & Finalization
Collecting internal and instructor feedback and refining all project elements.

### 8. Final Presentation
Preparing the final slides, documentation, and interactive prototype.
